EndSwapMove

Synopsis

This McMove attempts to reverse the order of atom types in a flexible linear heteropolymer. It was designed for use with diblock copolymers, but the algorithm can be used for any heteropolymer. The current version can only be used with flexible molecules, with no angle or dihedral potentials. (This constraint could be easily relaxed to allow homogeneous angle and dihedral potentials).

The Species identified by the speciesId must be a subclass of Linear, and may not be mutable.
