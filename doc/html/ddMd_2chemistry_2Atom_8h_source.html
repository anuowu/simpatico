473  /*
474  * Accessors for pseudo-members stored in separate arrays.
475  *
476  * An atom can be constructed only as an element of a parent AtomArray
477  * (see documentation for private default constructor). Some data owned
478  * by an Atom is stored in a set of private array that are allocated by
479  * its AtomArray. In each such array, the element associated with this
480  * Atom is indexed by the localId_ of this atom, without the least
481  * signficant bit (see documentation of the localId_ member). The bit
482  * shift operation "localId_ >> 1" in each of the following functions
483  * strips off the least signficant bit, which is used to store the isGhost
484  * flag, and returns the relevant array index. Each arrays that stores
485  * pseudo-member data is a private member of the parent AtomArray (e.g.,
486  * AtomArray::velocities_, AtomArray::masks_, etc.) that is accessible
487  * because Atom is a friend class of AtomArray.
488  */
