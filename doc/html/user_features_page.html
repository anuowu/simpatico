2.8 Parameter File Options

2.7 Data Analysis (Prev)         2.9 Restarting (Next)

These pages provide links to user documentation for features of each program that can be selected at run time by including appropriate polymorphic blocks in the parameter file. Such entries in the parameter file can be used to choose particular potential energies (interaction classes), molecular dynamics integrators or Monte Carlo moves, and data analysis operations. User documentation of the format for the parameter file for each such block is available through the links given below.
