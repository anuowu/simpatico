McMd::LinkPotentialImpl< Interaction > Class Template Reference

Detailed Description

template<class Interaction>
class McMd::LinkPotentialImpl< Interaction >

Template implementation of an BondPotential for links.

A LinkPotential implements the BondPotential interface, and uses an instance of a bond interaction class, but uses the LinkMaster of the parent System to identify linked atoms.
