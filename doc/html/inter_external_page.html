External Interactions

An external potential is a potential that acts on each atom, whose value depends only on the atomic position and atom type index.

All of the angle interaction classes implement a common interface, which is described here.

External Interaction Classes

  • LamellarOrderExternal - designed to induce layers
  • SlitExternal - 1D confinement to a slit
  • BoxExternal - 3D confinement to a box
  • OrthoBoxExternal - Generalized confinement to a box, channel or slit
