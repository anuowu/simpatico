Detailed Description

Analyzer that computes averages and block averages of a single Tensor.

This class evaluates the average of a sequence of Tensor values, and optionally writes block averages to a data file during the run. It is intended for use as a base class for classes that evaluate averages for specific tensor-valued physical variables.
