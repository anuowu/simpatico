Detailed Description

template<typename Data, typename Product>
class DdMd::AutoCorrAnalyzer< Data, Product >

Compute an autocorrelation function for a sequence of Data values.

This template works for Data types float or double, std::complex<real> with float or double real type, Util::Vector and Util::Tensor.
