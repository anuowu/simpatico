NvtIntegrator

Synopsis

NvtIntegrator implements a reversible Nose'-Hoover NVT (isothermal) integrator.

This integrator requires that the Util::EnergyEnsemble of the associated Simulation must be set to "isothermal". The target temperature is the temperature returned by the function Util::EnergyEnsemble::temperature().
